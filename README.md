# PagBank + EZZE Consórcios — Preparação Executiva

Site estático em HTML para GitHub Pages.

## Publicação
1. Envie `index.html` para a raiz do repositório.
2. Ative GitHub Pages em Settings > Pages.
3. Abra a URL pública do repositório.

## Objetivo
Material interno de preparação para reunião, com narrativa executiva, modelo de referência Sem Parar, produto EZZE/Âncora, papel da assessoria e simulação de comissão de 1%.
